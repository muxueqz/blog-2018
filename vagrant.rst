Vagrant是一个基于Ruby的工具，用于创建和部署虚拟化开发环境。它
使用Oracle的开源VirtualBox虚拟化系统，使用 Chef创建自动化虚拟环境。
 
 主页：[[http://vagrantup.com/\|]] 
 安装方法：
 sudo gem install vagrant 
 vagrant box add base http://files.vagrantup.com/base.box 
 mkdir vagrant 
 vagrant init 
 vagrant up
